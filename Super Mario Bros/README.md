# Super Mario Bros. 1-1 JavaFX Recreation

This project recreates the first two Mario course areas as a Java desktop application using JavaFX, FXML, Canvas rendering, and an MVC split.

Run it with:

```powershell
powershell -ExecutionPolicy Bypass -File .\Start-SuperMario.ps1
```

On Linux or macOS:

```bash
./Start-SuperMario.sh
```

Controls:

- `Left` / `Right`: move
- `Z`, `Space`, or `Up`: jump
- `X` or `Shift`: run
- `Down`: enter vertical pipes
- `R`: restart
- `1`: load World 1-1
- `2`: load World 1-2

Menus:

- `File > Exit`
- `Edit > Preferences`
- `Help > Controls`
- `Help > Rules`
- `Help > About`
